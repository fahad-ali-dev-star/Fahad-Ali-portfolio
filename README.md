# Fahad Ali Portfolio

Modern portfolio website built with React, TypeScript, and Vite.

## Features

- React + TypeScript + Vite
- Three.js 3D background effects
- Responsive design with Tailwind CSS
- Smooth animations and transitions

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Three.js
